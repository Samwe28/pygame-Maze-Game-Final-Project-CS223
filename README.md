# Pygame Maze Game

Pygame Maze Game is a 2D maze game built with Python and Pygame. The player moves through a randomly generated maze, avoids walls, and tries to reach the exit as quickly as possible. The game includes solo play, two-player mode, and an AI race mode.

## Features

- Randomly generated maze layout
- One-player mode
- Two-player race mode
- AI race mode
- Timer that tracks how long it takes to escape
- Best-time tracking for solo mode
- Start screen and win screen
- Keyboard movement controls
- Maze, player, timer, and scoreboard classes
